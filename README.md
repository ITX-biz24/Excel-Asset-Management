# Excel Asset Management

Excel and VBA-based asset management tool.

ExcelとVBAを使用した資産管理ツールです。

---

## 🇯🇵 日本語

### 概要

Excelで資産を管理するためのツールです。
VBAを使用して入力・検索・管理を効率化します。

### 動作環境

- Microsoft Excel
- Windows
- VBA (Visual Basic for Applications)
